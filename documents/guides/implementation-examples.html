📌 ケース2：Deploy to Vercel Skill

目的： アプリを Vercel に自動デプロイ
スクリプト数： 2個 (deploy.sh, deploy-codex.sh)
対応フレームワーク： 40+（Next.js, React, Vue, etc.）

デプロイフロー決定ツリー

デプロイ方法の自動選択： if (.vercel/project.json 存在 && git remote あり) { → git push デプロイ（推奨） } else if (Vercel CLI 認証済み) { → vercel deploy コマンド } else if (非認証環境・claude.ai) { → デプロイスクリプト実行 }
