# エグゼクティブ報告書 (Executive Reports)

このフォルダには、経営層向けのエグゼクティブ報告書を保存します。

## 推奨テンプレート

- [エグゼクティブ報告書](../templates/executive.html) - 経営層向けプレミアムデザイン
- [標準報告書](../templates/standard.html) - 汎用テンプレート

## 使用方法

1. 上記のテンプレートをブラウザで開く
2. 内容を編集する
   - エグゼクティブサマリー
   - KPIダッシュボード
   - 主要インサイト
   - リスク管理
3. ブラウザの印刷機能（Ctrl + P）からPDFとして保存
4. このフォルダに保存

## ファイル命名規則

推奨フォーマット: `YYYY-MM-executive-report.pdf` または `YYYY-QX-executive-report.pdf`

例:

- `2026-03-executive-report.pdf` (月次)
- `2026-Q1-executive-report.pdf` (四半期)
- `2026-annual-executive-report.pdf` (年次)

## 注意事項

- エグゼクティブ報告書は機密情報を含むことが多いため、適切に管理してください
- 配布前に承認プロセスを経ることを推奨します
- 必要に応じてウォーターマークや機密区分を追加してください
