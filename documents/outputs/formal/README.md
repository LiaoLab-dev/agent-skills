# 正式報告書 (Formal Reports)

このフォルダには、正式なビジネス文書・報告書を保存します。

## 推奨テンプレート

- [正式報告書](../templates/formal.html) - フォーマルな文書形式
- [標準報告書](../templates/standard.html) - 汎用テンプレート

## 使用方法

1. 上記のテンプレートをブラウザで開く
2. 内容を編集する
   - 表紙情報
   - 章立て構成
   - 承認欄
3. ブラウザの印刷機能（Ctrl + P）からPDFとして保存
4. このフォルダに保存

## ファイル命名規則

推奨フォーマット: `YYYY-MM-DD-document-name.pdf`

例:

- `2026-03-31-project-proposal.pdf` (提案書)
- `2026-03-31-annual-report.pdf` (年次報告書)
- `2026-03-31-technical-specification.pdf` (技術仕様書)

## 用途例

- 提案書
- 契約関連文書
- 監査報告書
- 技術仕様書
- 年次報告書
- 承認が必要な公式文書

## 注意事項

- 承認欄を必ず記入してください
- 管理番号を付与することを推奨します
- 版数管理を適切に行ってください
