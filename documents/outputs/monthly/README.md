# 月次報告書 (Monthly Reports)

このフォルダには、月次報告書の成果物を保存します。

## 推奨テンプレート

- [標準報告書](../templates/standard.html) - 最も汎用的なテンプレート
- [エグゼクティブ報告書](../templates/executive.html) - 経営層向け
- [正式報告書](../templates/formal.html) - フォーマルな文書

## 使用方法

1. 上記のテンプレートをブラウザで開く
2. 内容を編集する
3. ブラウザの印刷機能（Ctrl + P）からPDFとして保存
4. このフォルダに保存（ファイル名例: `2026-03-monthly-report.pdf`）

## ファイル命名規則

推奨フォーマット: `YYYY-MM-monthly-report.pdf`

例:

- `2026-03-monthly-report.pdf`
- `2026-04-monthly-report.pdf`
