# スプリント報告書 (Sprint Reports)

このフォルダには、アジャイル・スクラムのスプリント報告書を保存します。

## 推奨テンプレート

- [スプリント報告書](../templates/agile.html) - アジャイル・スクラム専用

## 使用方法

1. 上記のテンプレートをブラウザで開く
2. スプリント情報を編集する
   - スプリント番号と期間
   - 前スプリントのアクションアイテムフォローアップ
   - スプリント目標
   - ベロシティとメトリクス
   - バーンダウンチャート
   - ユーザーストーリー進捗
   - デイリースタンドアップハイライト
   - レトロスペクティブ（KPT）
   - 次スプリント計画
3. ブラウザの印刷機能（Ctrl + P）からPDFとして保存
4. このフォルダに保存（ファイル名例: `sprint-12-report.pdf`）

## ファイル命名規則

推奨フォーマット: `sprint-XX-report.pdf`

例:

- `sprint-12-report.pdf`
- `sprint-13-report.pdf`

## チェックリスト

スプリント報告書作成時の確認項目：

- [ ] 前スプリントのアクションアイテムがフォローアップされている
- [ ] スプリント目標が明確に記載されている
- [ ] バーンダウンチャートが更新されている
- [ ] ベロシティが正確に記録されている
- [ ] 全ユーザーストーリーのステータスが最新
- [ ] レトロスペクティブ（KPT）が記入されている
- [ ] 次スプリントの計画が含まれている
