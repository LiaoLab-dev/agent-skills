# 週次報告書 (Weekly Reports)

このフォルダには、週次報告書の成果物を保存します。

## 推奨テンプレート

- [週次報告書](../templates/compact.html) - コンパクトな1ページ版
- [ダッシュボード](../templates/dashboard.html) - メトリクス重視
- [モダン報告書](../templates/modern.html) - デザイン性重視

## 使用方法

1. 上記のテンプレートをブラウザで開く
2. 内容を編集する
3. ブラウザの印刷機能（Ctrl + P）からPDFとして保存
4. このフォルダに保存（ファイル名例: `2026-W13-weekly-report.pdf`）

## ファイル命名規則

推奨フォーマット: `YYYY-WXX-weekly-report.pdf`

例:

- `2026-W13-weekly-report.pdf` (2026年第13週)
- `2026-W14-weekly-report.pdf` (2026年第14週)
