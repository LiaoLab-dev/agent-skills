1 準備確認（2分）

以下がインストール済みか確認:

  • ✅ VS Code バージョン 1.84+
  • ✅ GitHub Copilot 拡張機能
  • ✅ Git & GitHub アカウント
  • ✅ 7zip または zip コマンド
# macOS / Linux での確認 which git which zip vs_code --version # Windows PowerShell での確認 Get-Command git Get-Command Compress-Archive
💡 すべてインストール済み？ → ステップ2へ
2 Skill ディレクトリ作成（2分）

~/.claude/skills に新しい Skill フォルダを作成:

mkdir -p ~/.claude/skills/my-first-skill cd ~/.claude/skills/my-first-skill # フォルダ構成: # my-first-skill/ # ├── SKILL.md # └── metadata.json
📝 注：my-first-skill をあなたの Skill 名に置き換えてください（例：team-colors-validator）
3 SKILL.md を作成（3分）

ファイル「SKILL.md」を作成して以下を貼り付け:

--- name: my-first-skill description: "Example Skill" metadata: author: "Your Name" version: "1.0.0" --- # My First Skill これは最初の Skill の例です。 ## When to Use - "Check format" - "Validate code" ## Usage このコマンドで動作: copilot --skill my-first-skill ## Output Checked: OK
✅ ファイルを保存（Ctrl+S）
4 metadata.json を作成（2分）

ファイル name「metadata.json」を作成して以下を貼り付け:

{ "name": "my-first-skill", "version": "1.0.0", "description": "Example Skill", "author": "Your Name", "license": "MIT", "triggers": ["Check format", "Validate code"] }
✅ ファイルを保存
5 zip ファイル作成（3分）

ターミナルで以下を実行:

# macOS / Linux cd ~/.claude/skills zip -r my-first-skill.zip my-first-skill/ # Windows PowerShell $source = "$env:USERPROFILE\.claude\skills\my-first-skill" $dest = "$env:USERPROFILE\.claude\skills\my-first-skill.zip" Compress-Archive -Path $source -DestinationPath $dest
🔍 確認: my-first-skill.zip が作成された？
6 VS Code でテスト（3分）

VS Code の Copilot Chat で以下のように入力:

@copilot Check format # または @copilot Validate code

期待される動作:

  • ✅ Copilot が Skill を検出
  • ✅「my-first-skill」が実行
  • ✅ Output: "Checked: OK" が表示
✨ おめでとう！最初の Skill が完成しました 🎉
⚠️ よくあるエラーと対処法
エラー 原因 対処法
Skill not found フォルダ位置が違う ~/.claude/skills/ にコピーしたか確認
SKILL.md parse error frontmatter の YAML 形式が不正 --- の前後に空行がないか確認
Zip not recognized zip コマンドがない 7zip / PowerShell Compress-Archive を使用
Copilot triggers 動作しない metadata.json の triggers が SKILL.md と不一致 両ファイルの triggers を同じキーワードに統一
