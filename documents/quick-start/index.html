1 インストール

最初に、Agent Skillsをインストールします：

npx skills add vercel-labs/agent-skills

このコマンドは、あなたのプロジェクトに6つの強力なスキルを追加します。

2 VS Codeで開く

プロジェクトをVS Codeで開きます：

code .

VS CodeにGitHub Copilot拡張機能がインストールされていることを確認してください。

3 スキルの確認

Copilot Chat でスキルが読み込まれたことを確認：

@skills list

6つのスキルがすべてリストに表示されるはずです。

✓ 完了！

これで、AI アシスタントが以下の機能を活用できるようになりました：

  • React ベストプラクティス（65個のルール）
  • コンポーネント設計パターン
  • Web デザインガイドライン（100個以上）
  • React Native スキル
  • Vercel 自動デプロイ（40+フレームワーク対応）
  • CLI トークン認証
