# 🔧 サーバー - ドキュメント提供インフラ

このフォルダには、ローカルドキュメントサーバーとその設定ファイルが含まれています。

## ファイル

- **server.js** - Node.jsドキュメントサーバー
- **package.json** - 依存関係の定義
- **start-server.bat** - Windows用サーバー起動スクリプト

## 使用方法

### サーバーの起動

```bash
# 方法1: バッチファイルを使用（Windows）
double-click start-server.bat

# 方法2: Node.jsコマンドで実行
node server.js

# 方法3: npm スクリプト
npm start
```

### アクセス

サーバーが起動したら、ブラウザで以下にアクセス：

```
http://localhost:3000
```

## 設定

server.jsは以下の設定で実行されます：

- **ポート**: 3000番
- **ルートディレクトリ**: documents/
- **キャッシュ**: 無効

## トラブルシューティング

| 問題 | 解決方法 |
|------|--------|
| ポート3000が使用中 | 他のアプリを終了するか、server.jsのポート番号を変更 |
| モジュール見つからない | `npm install` を実行 |
| パーミッションエラー | 管理者権限で実行 |
